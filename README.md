# Call Peace (콜 피스) 🎧

> **고객의 비언어적 폭력을 부드럽게 순화하여 상담원의 멘탈을 지켜주는 AI 서비스**

안녕하세요! 이 프로젝트는 감정 노동의 최전선에 있는 콜센터 상담원분들을 위해 시작되었어요. 고객의 거친 언어와 감정을 AI가 먼저 받아내고, 상담원에게는 순화된 표현으로 전달함으로써 그들의 마음을 보호하는 것이 목표랍니다.

---

## 💡 기획 의도 (Background)

콜센터 상담원분들은 매일 수많은 고객을 응대하며 욕설이나 고성 같은 '비언어적 폭력'에 노출되어 있어요. 이런 스트레스는 심각한 감정 소모와 트라우마로 이어지기도 하죠.

"혹시 고객의 화난 목소리나 거친 단어를 **기술적으로 필터링**해줄 수 있다면 어떨까?"

이런 고민에서 출발해, 상담원이 듣거나 보게 될 메시지를 실시간으로 분석하고 **순화된 언어**로 바꿔주는 백엔드 서버를 구축하게 되었습니다. 상담원이 상처받지 않고 업무에 집중할 수 있는 환경을 기술로 돕고 싶어요.

## ✨ 주요 기능 (Key Features)

사용자(상담원 또는 관리자)는 이 서비스를 통해 다음과 같은 기능을 이용할 수 있어요.

- **언어 순화 (Message Transformation)**
  고객이 보낸 욕설이나 공격적인 메시지를 입력하면, 문맥은 유지하되 **정중하고 부드러운 표현**으로 바뀐 내용을 받을 수 있어요.
  (예: "야 이 미친X아 당장 환불해!" → "고객님께서 환불 처리를 강력하게 원하고 계십니다.")

- **감정 시뮬레이션 (Emotion Simulation)**
  서비스 테스트를 위해 분노, 공포, 슬픔, 혐오 등 다양한 감정 상태의 고객 대사를 무작위로 생성해볼 수 있어요. 실제 상담 환경과 유사한 데이터를 제공합니다.

- **상담 세션 관리**
  상담 내역을 세션별로 저장하고 불러올 수 있어, 순화 전/후의 대화 내용을 비교하거나 기록할 수 있어요.

## 🛠 기술 스택 (Tech Stack)

복잡한 아이콘 대신 사용한 핵심 기술만 깔끔하게 정리했어요.

- **Python 3.12+**: 최신 파이썬 환경에서 개발했어요.
- **FastAPI**: 빠르고 간결한 비동기 웹 프레임워크를 사용했어요.
- **LangChain & LangGraph**: 유연한 LLM 에이전트 워크플로우를 구현했어요.
- **ChromaDB**: 벡터 데이터를 효율적으로 저장하고 검색하기 위해 사용했어요.
- **HuggingFace / OpenAI / Anthropic**: 상황에 맞는 최적의 AI 모델을 활용했어요.
- **uv**: 빠르고 안정적인 패키지 관리를 위해 도입했어요.
